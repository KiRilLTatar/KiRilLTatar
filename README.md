# Кирилл Татаринцев

Backend-разработчик (.NET).

Занимаюсь разработкой серверной логики, REST API и инфраструктуры сервисов. Основной интерес — архитектура backend-систем, производительность приложений и DevOps-подходы.

В настоящее время участвую в проектировании и разработке backend-экосистемы компании для оформления и генерации студенческих карт. Система реализована на C# (.NET) с использованием архитектуры REST API и микросервисного подхода. Инфраструктура контейнеризирована с использованием Docker; маршрутизация и проксирование входящих запросов реализованы через Nginx внутри изолированной Docker-сети. Также ведётся разработка внутренней CRM-части системы для управления заявками, обработки пользовательских данных и генерации документов.
